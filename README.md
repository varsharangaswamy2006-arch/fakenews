I# 🧠 Fake News OpenEnv FINAL

A Machine Learning + Open Environment (OpenEnv) based web application to detect whether a news article is FAKE ❌ or REAL ✅.

🚀 Live Demo
👉 https://huggingface.co/spaces/varsha0706/fakenewsopenenv

📌 Features
Fake News Detection using ML
TF-IDF + Logistic Regression model
OpenEnv RL-style environment
3 difficulty levels (easy, medium, hard)
Agent grading (accuracy-based)
Reward system with confidence bonus
Interactive UI using Streamlit
Evaluation and Metrics visualization
🧠 Tech Stack
Python
Streamlit
Scikit-learn
NumPy
Matplotlib
PyYAML
⚙️ How it Works
🔹 Model
Text → TF-IDF Vectorization
Logistic Regression classifier
Predicts FAKE (0) or REAL (1)
🔹 OpenEnv System
Environment simulates tasks
3 difficulty levels:
Easy
Medium
Hard
Reward system:
Correct → +1
Wrong → -1
Confidence bonus added
🔹 Evaluation
Accuracy score
Average reward
Confusion matrix
